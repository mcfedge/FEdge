# Forward Edge-AI Brand Guidelines

## Primary Brand Colors

Use these exact hex values for all brand materials:

- **Purple Primary Light**: `#8275b1` - Main brand color, use for headers and key elements
- **Purple Primary Dark**: `#574b7c` - Secondary brand color, use for accents and contrast
- **Gray Light**: `#999999` - Use for secondary text and subtle elements
- **Gray Dark**: `#676767` - Use for body text and borders

## Typography

### Font Family
**Roboto** - Use for all text elements

### Font Sizes (in points)
- **Header 1**: 28pt
- **Header 2**: 18pt
- **Date**: 14pt
- **Body**: 16pt
- **Footer**: 12pt

## Gradients

### Gradient Specifications
- **Type**: Linear gradient ONLY (never radial)
- **Angle**: 0° (horizontal)
- **Color stops**:
  - Primary color at 0%
  - Mid-point at 50%
  - Secondary color at 100%

### Available Gradient Combinations
Based on the brand colors, use these gradient patterns:

1. **Purple Gradient**: `#574b7c` (0%) → mid-blend (50%) → `#8275b1` (100%)
2. **Gray Gradient**: `#676767` (0%) → mid-blend (50%) → `#999999` (100%)

**Important**: Do not create custom gradients. Use only the gradient combinations specified here.

## Logo Assets

Logos are available in the `assets/` directory:

### Available Logos
1. **fe_logo_dark.png** - Forward Edge-AI logo (dark version)
   - Use on light backgrounds
   - Primary company logo
   
2. **fe_logo_white.png** - Forward Edge-AI logo (white version)
   - Use on dark or colored backgrounds
   - Ensures visibility on purple gradients

3. **isidore_logo_dark.png** - Isidore logo (dark version)
   - Product/division branding
   - Use on light backgrounds

### Logo Usage Guidelines
- Maintain aspect ratio when resizing
- Provide adequate clearspace around logos
- Use dark logos on light backgrounds, white logos on dark/colored backgrounds
- Logos should typically appear in headers or title slides

## Color Application Examples

### For Presentations
- **Title slides**: Use purple gradient background (#574b7c to #8275b1) with white logo
- **Content slides**: White or light gray background with purple headers
- **Accent elements**: Use purple primary colors for bullets, dividers, and highlights

### For Documents
- **Headers**: Purple Primary Light (#8275b1) or Dark (#574b7c)
- **Body text**: Gray Dark (#676767)
- **Highlights/callouts**: Purple Primary colors
- **Footers**: Gray Light (#999999) at 12pt

### For Spreadsheets
- **Header rows**: Purple Primary Dark (#574b7c) background with white text
- **Alternating rows**: Light gray (#999999) for readability
- **Important cells**: Purple Primary Light (#8275b1) for emphasis
- **Charts**: Use primary purple colors consistently
